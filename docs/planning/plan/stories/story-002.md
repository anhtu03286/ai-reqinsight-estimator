# Story 002 — View & Approve AI Results

## Metadata
- **Epic:** EPIC-02 AI Core + EPIC-03 Collaboration
- **Priority:** P0
- **Status:** Todo

## User story

As a **PM**, I want to **view all Gaps/Risks/Clarifications detected by AI, edit them if needed, and approve each item**, to control quality before generating a quotation.

## Acceptance Criteria

- [ ] Dashboard displays results categorized by type and severity
- [ ] Click on an item → navigates to the original text in the document
- [ ] Can edit AI content; edit history is saved
- [ ] Approve/Reject each item, following role permissions
- [ ] Only Approved items are included in the quotation
