# Reviews

Directory containing **Reviewer Agent output** (Agent 05).

| File | Description |
|------|-------------|
| `review.md` | Default review result: **PASS** or **FAIL** + reasons |

## Multiple PRs

Each review can be placed per PR, e.g. `docs/planning/reviews/review-pr-42.md`.

## When creating new files

- Agent definition: [reviewer-agent.md](../.ai-factory/agents/reviewer-agent.md)
- Review template: [review-template.md](../.ai-factory/templates/review-template.md)

Directory is empty until a PR has been reviewed (README only).
