# Product

Directory containing **Product Agent output** (Agent 01).

| File | Description |
|------|-------------|
| `prd.md` | Full PRD — problem, users, goals, modules, workflows, AC |
| `business-rules.md` | Detailed business rules |
| `success-metrics.md` | Success metrics, baseline / targets |

## When creating new files

- Agent definition: [product-agent.md](../.ai-factory/agents/product-agent.md)
- Templates: [prd-template.md](../.ai-factory/templates/prd-template.md), [business-rules-template.md](../.ai-factory/templates/business-rules-template.md), [success-metrics-template.md](../.ai-factory/templates/success-metrics-template.md)

The files above are generated when the Product Agent runs on a project — the original template leaves this directory empty (README only).
