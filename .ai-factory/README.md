# AI Factory

The agent framework, templates, and reference documentation for AI-assisted development.

| Directory | Purpose |
|-----------|---------|
| `agents/` | Role definitions for each agent |
| `templates/` | Skeletons for PRD, business rules, metrics, architecture, module/DB/API, epic, story, task, review, lesson |
| `skills/` | Reusable skills / checklists for agents |
| `knowledge/` | Domain knowledge, glossary, ADR |
| `workflows/` | End-to-end process descriptions (e.g. `knowledge-loop.md`) |

Artifact flow: idea → `docs/planning/product/` → `docs/planning/architecture/` → `docs/planning/plan/` (epics, stories, tasks) → code/PR → `docs/planning/reviews/`.
